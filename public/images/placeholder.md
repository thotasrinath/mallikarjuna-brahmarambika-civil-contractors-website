# Image Placeholders

This folder contains placeholder images that can be used temporarily until actual project photos are available.

## Recommended Image Sizes
- **Hero images**: 1920x1080 pixels
- **Portfolio thumbnails**: 800x600 pixels  
- **Service icons**: 400x400 pixels
- **Team photos**: 600x600 pixels

## Color Scheme Used in Placeholders
- Blue gradients (#0ea5e9 to #0369a1) - Primary brand colors
- Orange gradients (#f97316 to #ea580c) - Accent colors
- Green (#10b981) - Success indicators
- Gray gradients - Neutral backgrounds

## Adding Actual Images
1. Replace placeholder images with actual project photos
2. Maintain consistent sizing and aspect ratios
3. Optimize images for web (compressed, WebP format recommended)
4. Add alt text for accessibility
