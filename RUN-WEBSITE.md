# Running the Nuxt Website

## Quick Start Instructions

### Method 1: Using Command Prompt/PowerShell

1. Open Command Prompt or PowerShell
2. Navigate to the project directory:
   ```cmd
   cd C:\Srinath\Projects\mallikarjuna-brahmarambika-constructions\nuxt-website
   ```

3. Install dependencies (if not already installed):
   ```cmd
   npm install
   ```

4. Start the development server:
   ```cmd
   npm run dev
   ```

5. Open your browser and go to: http://localhost:3000

### Method 2: Using the Batch File

1. Navigate to the `nuxt-website` folder
2. Double-click `start-website.bat`
3. Wait for the server to start
4. Open your browser to http://localhost:3000

### Method 3: Using File Explorer

1. Open Windows Explorer
2. Navigate to: `C:\Srinath\Projects\mallikarjuna-brahmarambika-constructions\nuxt-website`
3. Hold Shift + Right-click in the folder
4. Select "Open PowerShell window here"
5. Run: `npm run dev`
6. Visit: http://localhost:3000

## Troubleshooting

### If you get "package.json not found":
- Make sure you're in the correct directory
- Verify `package.json` exists in the current folder

### If npm commands fail:
- Ensure Node.js is installed: `node --version`
- Update npm: `npm install -g npm@latest`

### If port 3000 is already in use:
- Use a different port: `npm run dev -- --port 3001`
- Then visit: http://localhost:3001
